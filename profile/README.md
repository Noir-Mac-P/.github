## 📖 Overview

<a href="#noir">Noir</a> is a Safari extension for Mac that automatically applies a dark visual style to websites that don't natively support dark mode — giving every website a dark background and adjusted text colors when macOS Dark Mode is active, eliminating the jarring contrast between the dark Mac interface and bright white websites.

The <a href="#noir">automatic dark mode application</a> works without configuration — when Noir is installed and macOS Dark Mode is enabled, websites that don't have their own dark mode implementation receive Noir's intelligent dark style automatically. The extension analyzes each page's colors and applies a dark transformation that inverts and adjusts the color scheme while attempting to preserve the site's visual identity rather than applying a crude color inversion. Images and media are handled separately from text and background elements to avoid the unnatural appearance of color-inverted photographs. The <a href="#noir">intelligent color transformation</a> distinguishes Noir from simple dark mode extensions — rather than inverting all colors uniformly, Noir applies a considered transformation that darkens backgrounds and adjusts text and UI element colors while maintaining legibility and visual coherence. The result reads as a dark-themed version of the site rather than as a color inversion filter.

The <a href="#noir">per-site customization</a> configures dark mode settings for individual websites — brightness and contrast adjustments for sites where the automatic transformation needs refinement, and whitelisting for sites that already have their own dark mode that Noir should not override. The <a href="#noir">system Dark Mode sync</a> activates and deactivates automatically with macOS Dark Mode — when the Mac switches to Light Mode, Noir deactivates; when it switches to Dark Mode, Noir activates — no manual toggling required.</p>

<a href="#noir">Scheduled dark mode activation</a> sets time-based dark mode schedules independent of the system setting — enabling dark browsing in the evening without switching the entire Mac to dark mode, or synchronizing with the system's automatic light and dark schedule. <a href="#noir">Brightness and contrast controls</a> adjust the darkness level of the applied dark style — useful for sites where Noir's default transformation is too dark or not dark enough, or for personal preference in dark intensity.

<a href="#noir">Safari extension architecture</a> integrates Noir directly into Safari as a native extension — no separate application running, no proxy, no traffic interception. Noir operates within Safari's extension sandbox, applying CSS transformations to page rendering. <a href="#noir">Support for all Safari contexts</a> applies dark mode across regular browsing, private browsing, and across all tabs — dark mode is consistent throughout the browsing session.</p>

## ❓ FAQ

<details>
<summary>Does Noir work automatically or does it need configuration?</summary>

Noir works automatically — when installed with macOS Dark Mode active, dark mode applies to websites without dark mode support automatically.

</details>

<details>
<summary>Does Noir affect websites that already have dark mode?</summary>

Websites with good native dark mode implementations can be whitelisted so Noir does not apply its transformation to them.

</details>

<details>
<summary>Does Noir invert colors?</summary>

Noir applies an intelligent dark transformation rather than simple color inversion — backgrounds darken and UI colors adjust while images and media are handled to preserve their natural appearance.

</details>

<details>
<summary>Does Noir sync with macOS Dark Mode?</summary>

Yes. Noir activates and deactivates automatically when macOS switches between Dark and Light Mode.

</details>

<details>
<summary>Can I schedule Noir to activate at certain times?</summary>

Yes. Scheduled activation enables time-based dark mode independent of the system appearance setting.

</details>

<details>
<summary>Does Noir work in private browsing?</summary>

Yes. Noir applies dark mode in both regular and private Safari browsing.

</details>

<details>
<summary>Is Noir a subscription or one-time purchase?</summary>

One-time purchase from the Mac App Store.

</details>

<details>
<summary>Can I adjust the darkness level per website?</summary>

Yes. Per-site brightness and contrast controls adjust the transformation intensity for individual websites.

</details>

<details>
<summary>Does Noir work on Apple Silicon?</summary>

Yes. Noir is a Safari extension and works on Apple Silicon Macs.

</details>
